🎓 Grade Calculator

A simple Java Console Application that calculate a student's Total Marks, Average, and Grade based on marks entered for multiple subject.

📌 Features
Accepts student name
Accept number of subjects
Take marks for each subject
Calculates:
✅ Total Marks
✅ Average Marks
✅ Grade (A–F)
Simple and beginner-friendly Java project

🛠️ Technologies Used
Java
Scanner Class (User Input)
Conditional Statements (if-else)
Loops (for loop)
